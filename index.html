How To Play
Guess the COLOR CODE in six tries.
The left pallete shows the target color. The right pallete shows the color of your last guess.
Each guess must be a six-digit hex color code made up of
3 pairs of digits each representing Red, Green, and Blue components of the color in order ranging from 00 to FF. Hit the enter button to submit.
After each guess, the color of the tiles will change to show how close your guess was to the hex code.
